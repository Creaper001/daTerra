# Zeus API

## Informação

Api de base para fornecer as informaçoes e acesso ao sistema web **daTerra**.

## Tecnologia

Foi utilizado o framework [**Adonis**](https://adonisjs.com/) para montar a estrutura base e simplificar a padronização.

## Banco de dados

Vamos utilizar um plato gratuito da plataforma [**PlanetScale**](https://planetscale.com/) para gerenciar um banco de dados MySql de maneira Serveless (Sem Servidor).

## Hospedagem

A Hospedagem vai ser feito na plataforma Cloud do [**Heroku**](https://www.heroku.com/) em sua versão gratuita.

## Iniciando projeto

### Oficial

      yarn install  
      yarn build
      yarn start

Ou via **NPM**

      npm install
      npm run build
      npm run start

### Dev

      yarn install  
      yarn dev

Ou via **NPM**

      npm install
      npm run dev
